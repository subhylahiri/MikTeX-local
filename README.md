MikTeX-local
============

Bib files, macro lists etc. for including in all other LaTeX projects

## CTAN packages

You'll need
* a4wide
* ntgclass
* natbib
* url
* xcolor
* mptopdf
* beamer
* pgf
* ms
* sansmathaccent
* filehook
* appendixnumberbeamer
* etoolbox
* adjustbox
* collectbox
* tcolorbox
* environ
* trimspaces
* type1cm
* fp
* caption
* paralist
* cancel
* appendix
